
# 轮毂电机控制接口说明文档

## 一、控制模式说明

1. **默认控制模式**：位置环控制（BM_MODE_POSITION）
2. **模式切换规则**（通过 P/D 参数自动判断，优先级从高到低）：
   - P 参数（P_N_Rad）≠ 0 → 位置环控制
   - P 参数 = 0 且 D 参数（D_N_Rad_s）≠ 0 → 速度环控制
   - P 参数 = 0 且 D 参数 = 0 → 电流环控制（暂不使用）

## 二、模式切换约束（重要）

1. 位置环运动执行期间，**禁止切换控制模式**
2. 电机到达目标位置后，**可切换为速度环模式**控制

## 三、MIT控制数据发送接口（速度环/位置环通用）

### 函数原型

int Send_MIT_PD_Control_Data(std::shared_ptr<Device_class> Device_P, float Rad, float Speed_Rad_S, float Force_N, float P_N_Rad, float D_N_Rad_s);

### 参数说明

| 参数名            单位        位置环      速度环          说明
|---------------|----------|----------|----------|-----------------|
| Device_P      | -              必传       必传        设备句柄
| Rad           | 度             有效        无效       目标位置值
| Speed_Rad_S   | 度/秒          无效        有效        目标速度值（需转RPM）
| Force_N       | N             有效        无效        力矩值
| P_N_Rad       | -             有效        无效        位置环P参数
| D_N_Rad_s     | -             无效        有效        速度环D参数

## 四、使用流程建议

1. 上电默认进入**位置环控制**
2. 位置环运动过程中，保持模式不变
3. 位置到达后，将 P 参数置 0、D 参数置非 0，切换为**速度环控制**
